# BoilerPlate AfterJs, Styled-components and Heroku

## Start your project

Close the repository

`git clone https://github.com/nicholasess/boilerplate-afterjs-styledcomponents-heroku projectname`

Enter in path project

`cd projectname`

To start
`yarn install`

Access localhost
`http://localhost:3000`

To Build
`yarn build`
